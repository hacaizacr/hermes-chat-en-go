# 🚀 Hermes: Real-Time Chat System

**Hermes** es una aplicación de chat en tiempo real diseñada con una arquitectura cliente-servidor, enfocada en la eficiencia del manejo de mensajes y la gestión de salas dinámicas. Este proyecto fue desarrollado para explorar la concurrencia y la comunicación bidireccional en entornos de red.

## 🛠️ Tecnologías Utilizadas
* **Backend:** [Go (Golang)](https://golang.org/) - Implementación de servidores eficientes.
* **Frontend:** HTML5 y CSS3 - Interfaz de usuario minimalista.
* **Protocolo:** WebSockets / Net Package para comunicación en tiempo real.

## ✨ Características Principales
* **Lógica de Salas (Rooms):** Gestión de múltiples salas de chat independientes.
* **Manejo de Concurrencia:** Uso de *Goroutines* y *Channels* para procesar mensajes de forma simultánea.
* **Arquitectura Limpia:** Código organizado para facilitar el mantenimiento y escalabilidad.

## 🚀 Cómo Ejecutar
1. Asegúrate de tener **Go** instalado.
2. Clona el repositorio y entra a la carpeta.
3. Ejecuta: `go run main.go`
